## 📐 Design System Overview

### **Responsive Units**
- **rem** - Relative to root font size (16px default)
- **em** - Relative to parent font size
- **%** - Relative to container
- **vw/vh** - Viewport units
- **clamp()** - Fluid sizing between min/max

### **Spacing Scale**
- xs (4px), sm (8px), md (16px), lg (24px), xl (32px), 2xl (48px), 3xl (64px)

### **Font Sizes**
- xs (10-12px), sm (12-14px), base (14-16px), md (16-18px), lg (18-24px), xl (24-32px), 2xl (28-42px), 3xl (32-48px)

## 📚 Document Naming Convention

- **PascalCase** for file names (e.g., `ResponsiveDesignGuide.md`)
- **Descriptive names** that indicate content (e.g., `QuickReference.md`)
- **Avoid abbreviations** unless widely known (e.g., `CSS`, `API`)
